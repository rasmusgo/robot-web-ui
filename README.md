# Robot Web UI
TypeSript + node.js + express + react + webpack.

## Building once and running
Run this in a terminal:

    yarn install --frozen-lockfile
    yarn build-backend
    yarn build-frontend
    yarn start

## Building and running in watch mode
One terminal for each watch command is needed.

    yarn install --frozen-lockfile
    yarn watch-backend
    yarn watch-frontend
    yarn watch-node
